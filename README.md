# Automated Text Summarization using Golf Optimization Algorithm (GOA)

## Description
This project implements an extractive text summarization system using the Golf Optimization Algorithm (GOA). The system selects important sentences from a document to generate a concise summary.

## Technologies Used
- Python
- Tkinter
- SQLite
- GOA Algorithm

## Features
- Upload text documents
- Generate optimized summaries
- Store summaries in database
- GUI-based interface

## Files
- main_gui.py → GUI implementation
- goa_logic.py → GOA algorithm
- db.py → Database functions
- summaries.db → Database storage
- view_data.py → View stored summaries

## How to Run

1. Install Python
2. Install required libraries
3. Run:

```bash
python main_gui.py
```
